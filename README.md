# 👾 My portfolio website

I created this website to learn vanilla HTML/CSS/JS. My project is inside a Vite project using a github workflow for build and GitHub Pages deployment. For this website, I followed a step-by-step tutorial on youtube by [How to Web Dev](https://www.youtube.com/@HowtoWebDev) and added some of my own work. For the 3D assets, I searched for .gltf files on [SketchFab](https://sketchfab.com/feed). The 3D library used is [THREE.js](https://threejs.org/).
